# MQ
基于 [JMCCC](https://github.com/to2mbn/JMCCC) 的Minecraft启动器

## License

本项目采用协议 [LGPL](LICENSE.txt).

( JMCCC使用协议 [MIT license](https://to2mbn.github.io/jmccc/LICENSE.txt))

## Features

- 能够启动所有版本的 Minecraft Java版 客户端。

- 下载并自动安装任何版本的 Minecraft Java版 客户端。

- 使用JavaFX实现可视化界面交互。

## 后记
